Purpose of each file:

Superinfection code refers to MBRD-SI model implementation, coinfection code refers to MBRD-CI model implementation.

"coinfect_pointsource_b12thres_alpha12": beta_12 threshold for endemicity given various alpha12 values

"coinfect_pointsource_b12varied": point source spread for varying b12 values

"superinfect_pointsource_sigmavaried": point source spread for varying sigma values

"superinfect_pointsource_timevaried" and "coinfect_pointsource_timevaried": point source spread for varying outbreak timings
