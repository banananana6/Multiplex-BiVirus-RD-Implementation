Purpose of each file:

Superinfection code refers to MBRD-SI model implementation, coinfection code refers to MBRD-CI model implementation.

"superinfect_equilibrium" and "coinfect_equilibrium": calculates equilibrium values

"superinfect_lattice_patterngen" and "coinfect_lattice_patterngen": generates a Turing pattern on lattice networks

"superinfect_parameter_sensitivity" and "coinfect_parameter_sensitivity": analyzes cross-diffusion parameter sensitivity for the example Turing patterns

"superinfect_ws_patterngen" and "coinfect_ws_patterngen": generates a Turing pattern on Watts-Strogatz (small-world) networks

"superinfect_parameter_sensitivity_sigma=2": pattern-forming parameter regime for sigma=2 (Claim 1 of companion paper)

"coinfect_parameter_sensitivity_b_turnedoff": pattern-forming parameters when beta_10, beta_02, and beta_12 are turned off (Claim 3 of companion paper)

"coinfect_parameter_seensitivity_dC": pattern-forming parameters for varying dC (coinfect group's diffusion) values
